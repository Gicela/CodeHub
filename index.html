Codehub

Code hub is a place to code, share, and learn from one another. At Code hub, we will be working on a set project, but you are welcome to bring your own. If you are developing for the web and wish to exchange knowledge and skills, come and join us.

Is Code Hub for you?

Freelance designers, developers, engineers and anyone who makes stuff for the web are welcome.

What are we learning at Code Hub?

We will be touching on all the technologies that we use in our daily work. These include, HTML, CSS, JavaScript, JQuery, Mobile Web, PHP, MySQL, APIs, JSON, Open Data, Python, etc.

Where do we meet?

We meet the first Monday of the month (venue to be determined) for a morning session, from 9:30 - 1:30pm

With a 30min talk/discussion of a learning topic. The rest of the time is for hacking. We will continue to collaborate remotely outside the fixed session.
